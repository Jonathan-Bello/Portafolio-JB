This was a small personal project built for a graduation celebration. It was created in only a few hours so the invitation could be shared quickly.

The page included schedule, date, scroll animations and a carousel of event images. Technically it was simple, but emotionally it mattered because it helped someone close to me celebrate an important achievement.

It represents the personal side of development: sometimes a small website is valuable because it appears exactly when someone needs it.
