The GDD CSS website was born from necessity and conviction. Citadel of the Solar Souls was too large to live only in my head or in scattered conversations. It needed serious documentation.

Instead of writing a traditional document, I built the Game Design Document as a website. The goal was to create a public, navigable and understandable place for the team, advisors and anyone interested in the project.

The site documents gameplay systems, progression, interface, narrative, references, team and a playable Godot web demo. It became the map of the project: a living technical document for a game where CSS, AI, exploration and learning had to share the same language.
