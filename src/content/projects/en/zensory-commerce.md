Zensory was a focused landing page for a course, deployed on a subdomain and connected to Stripe Checkout.

The payment flow was intentionally simple: collect the necessary user data, create a secure Stripe session and redirect to checkout.stripe.com. That avoided building a custom payment page while keeping the transaction safe and practical.

For my portfolio, it represents a lightweight e-commerce pattern: one landing page, one product, one payment flow and a clear technical implementation.
