## Context

This project comes from my master's thesis about an educational CSS video game with AI-assisted code evaluation.

## Role

I led the research, proposal design, technical documentation, and prototype development, combining web technologies, game development, backend, and applied AI.

## Results

The prototype was evaluated with 50 students. The average score improved from 5.84 to 8.12, and the SUS usability score reached 81.4 points.
