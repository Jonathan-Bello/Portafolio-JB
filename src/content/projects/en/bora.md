Bora Boutique was the first website I remember building from scratch for a client. The design was already provided in Figma, but the implementation was fully mine.

It mattered to me because I could use the design criteria I had learned: consistent spacing, coherent icons, color system, smooth navigation and subtle animations. It was not only about making the page work; it was about making it feel intentional.

The client later stopped using the original production domain, but the project remains meaningful as one of my first complete client builds.
