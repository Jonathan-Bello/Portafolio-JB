Ads Atelier was a visually focused project for a jewelry brand. The challenge was not a large system, but achieving a premium feeling through restrained layout, image treatment and scroll details.

I built the site from scratch with Astro and Tailwind CSS, using scroll-based visual effects and a clean structure aligned with the design. The most technical part was integrating Instagram content, which required learning how the social API and temporary tokens worked.

For my portfolio, this project represents aesthetic implementation: taking a luxury design and translating it into a fast, polished website.
