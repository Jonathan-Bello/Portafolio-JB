Surima Sustentable Riviera Maya fue una landing de alcance acotado y entrega rápida. El diseño fue entregado por el equipo y la necesidad principal era construir el sitio en poco tiempo, con una presentación clara y visual.

El proyecto no llegó a consolidarse con un dominio propio, pero quedó publicado en Netlify como una versión funcional de la landing.

## Desarrollo y reto técnico

Construí la landing desde cero con Astro, integrando banner principal, secciones de propuesta, valores, contenido comercial, carrusel y footer. Fue un proyecto sencillo en alcance, pero útil para seguir puliendo una base de trabajo más rápida con Astro y Tailwind.

También fue la primera vez que probé Astro 5 en beta dentro de un proyecto pequeño. No era necesario aprovechar funciones avanzadas, pero la landing era un buen espacio para probar la versión sin comprometer un sistema grande.

El reto más visible fue el carrusel y la velocidad de entrega. El proyecto debía salir rápido, así que la prioridad fue construir una landing limpia, responsive y fácil de revisar, sin sobredimensionar la solución.

## Tecnologías utilizadas

1. Astro.js
2. Tailwind CSS
3. Swiper
4. Netlify

## Resultado

El sitio quedó como una landing comercial breve, visual y responsive. Para mi portafolio, Surima representa una entrega rápida con Astro, útil para mostrar cómo una base técnica ya dominada permite resolver proyectos pequeños en pocos días sin sacrificar presentación.
