Surima fue una landing page para comunicar contenido comercial de marca con apoyo de secciones visuales.

## Desarrollo y reto técnico

Desarrollé la landing, integré carruseles y preparé el despliegue en Netlify.

El reto fue usar carruseles y contenido visual sin sacrificar rendimiento ni claridad de lectura.

## Tecnologías utilizadas

1. Astro.js
2. Tailwind CSS
3. Swiper
4. Netlify

## Resultado

El sitio presenta la propuesta comercial de forma visual, responsive y fácil de recorrer.
