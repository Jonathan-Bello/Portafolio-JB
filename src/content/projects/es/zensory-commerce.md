## Contexto

Zensory fue una landing page orientada a la venta de un producto digital. El reto fue integrar una experiencia simple de compra, confirmación y comunicación posterior al pago.

## Solución

El proyecto usa Astro para la interfaz, Stripe Checkout para el pago y Nodemailer para el envío de correos de confirmación.

## Lo que demuestra

Es un caso útil para mostrar integraciones reales entre frontend, pagos, backend ligero y comunicación transaccional.
