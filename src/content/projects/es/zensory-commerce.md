Zensory fue una landing page para un curso de manifestación con sonido. El sitio presenta el curso, sus módulos y un flujo de compra digital.

## Desarrollo y reto técnico

Desarrollé la landing, integré Stripe Checkout y preparé el flujo de confirmación posterior al pago mediante correo.

El reto fue cerrar correctamente el ciclo de compra: iniciar Checkout, confirmar sesiones pagadas y automatizar el correo posterior con Nodemailer.

## Tecnologías utilizadas

1. Astro.js
2. React
3. Tailwind CSS
4. Stripe
5. Nodemailer
6. Netlify

## Resultado

El proyecto muestra una integración real entre frontend, pagos, backend ligero y comunicación transaccional para un producto digital.
