Bora Boutique fue una de las primeras páginas que desarrollé desde cero dentro de YGU. El diseño ya venía trabajado en Figma, como ocurría con muchos proyectos de la agencia, pero esta fue una de las primeras veces en las que me dijeron directamente: “tú haz esta landing”.

Eso hizo que el proyecto fuera especial. Pude aplicar criterios que venían de mi formación y de mi interés por la interfaz: márgenes consistentes, coherencia de iconografía, colores, sistema visual, navegación cuidada y animaciones que hicieran que la experiencia no se sintiera plana.

## Desarrollo y reto técnico

Desarrollé la estructura frontend del sitio, sus secciones principales y la integración con rutas de contacto o reservación. El proyecto usó Gatsby con TypeScript y Sass, lo que me permitió organizar estilos por bloques y trabajar una landing visual con mayor control.

El reto principal no fue una integración compleja, sino asumir el proyecto desde cero y llevar el diseño a código con fidelidad. Quería que la página no solo funcionara, sino que también se sintiera bien construida: con transiciones, entradas visuales, sliders, assets cuidados y una estructura clara para una marca turística.

También fue una experiencia importante porque me permitió ganar confianza creando una landing completa por mi cuenta. Aunque después el cliente dejó de usar el dominio principal y el sitio quedó conservado en un dominio provisional de Netlify, el proyecto sigue siendo una referencia de esa etapa inicial en la que empecé a construir sitios completos con más seguridad.

## Tecnologías utilizadas

1. Gatsby
2. React
3. TypeScript
4. Sass
5. React Slick
6. Hero Slider
7. Google Tag Manager

## Resultado

El sitio quedó como una landing visual para una marca hotelera boutique, con secciones comerciales, flujo hacia reservaciones y una implementación alineada al diseño original. Para mi portafolio, Bora representa una de mis primeras experiencias creando una página completa desde cero y cuidando tanto estructura como detalle visual.
