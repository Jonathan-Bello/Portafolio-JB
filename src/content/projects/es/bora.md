Bora fue un sitio web desarrollado con Gatsby para presentar una marca de hotelería boutique, con una experiencia visual orientada a comunicar ambiente, servicios y reservaciones.

## Desarrollo y reto técnico

Desarrollé la base frontend del sitio, la estructura de páginas y componentes de presentación. El proyecto usa Gatsby con TypeScript y Sass, lo que permitió separar estilos por secciones y construir una experiencia visual más controlada.

El reto estuvo en trabajar una landing con carga visual fuerte sin perder claridad. Sliders, assets, navegación y tracking tenían que convivir con una salida estática ligera.

## Tecnologías utilizadas

1. Gatsby
2. React
3. TypeScript
4. Sass
5. React Slick
6. Hero Slider
7. Google Tag Manager

## Resultado

El sitio quedó como una presencia web clara para una marca hotelera, con secciones visuales, ruta de reservaciones y una base técnica mantenible.
