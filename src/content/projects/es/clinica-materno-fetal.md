Clinica Materno Fetal pertenece a una etapa mas reciente de mi trabajo en IGHU, una etapa donde ya existian nuevas politicas de diseño, nuevos patrones de desarrollo y una forma mas madura de construir sitios institucionales.

No fue un proyecto especialmente complejo, y justo por eso tiene otro tipo de valor: muestra como, despues de tantos sitios, ciertas decisiones empiezan a volverse naturales. La arquitectura, los componentes reutilizables, los iconos, los llamados a la accion y la estructura de servicios ya no se sienten como problemas aislados, sino como piezas de un sistema que se puede levantar con mayor velocidad.

## Desarrollo y reto tecnico

Desarrollé el sitio con Astro, React, Tailwind CSS y Heroicons como parte de la iconografia estandar que se ha ido consolidando en los proyectos de IGHU. Aunque no era una landing page, muchas secciones compartian patrones visuales, lo que permitio reutilizar componentes y mantener una estructura consistente.

La parte mas destacable fue el catalogo de servicios y estudios. La clinica necesitaba presentar informacion medica de manera clara, ordenada y confiable, con accesos para que las personas pudieran agendar una cita. Por peticion del cliente, los llamados a la accion se dirigieron hacia WhatsApp, ya que ese canal les permitia administrar mejor la comunicacion con pacientes.

El componente mas retador fue el modal del equipo medico. No era una dificultad extrema, pero si requeria cuidar que la informacion de los doctores se mostrara bien, que la interaccion fuera limpia y que el sitio mantuviera una presencia profesional.

## Tecnologias utilizadas

1. Astro.js
2. React
3. Tailwind CSS
4. Heroicons

## Resultado

El resultado fue un sitio medico sencillo, claro y funcional. No trabajé con datos sensibles ni con formularios clinicos complejos; la prioridad fue presentar informacion de manera correcta, generar confianza y facilitar el contacto.

Para mi, este proyecto representa una etapa donde el oficio empieza a sentirse mas fluido: menos lucha contra la herramienta y mas claridad para resolver con arquitectura, comunicacion y buenos patrones de interfaz.
