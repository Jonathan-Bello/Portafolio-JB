## Contexto

La clínica requería un sitio institucional claro para pacientes, con servicios, información médica y canales de contacto.

## Mi rol

Desarrollé la experiencia web y organicé el contenido para que fuera legible, confiable y fácil de navegar desde móvil.

## Reto técnico

El proyecto debía publicarse dentro de una ruta base del dominio, por lo que fue necesario cuidar rutas, assets, metadata y navegación.

## Resultado

El sitio ofrece una presencia profesional para una clínica especializada, priorizando claridad, confianza y contacto.
