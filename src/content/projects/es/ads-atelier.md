Ads Atelier necesitaba una presencia web para comunicar una marca de joyería fina con un enfoque visual, editorial y comercial.

## Desarrollo y reto técnico

Desarrollé el sitio como proyecto web desde cero, estructurando las secciones principales, la navegación, el catálogo visual y los puntos de contacto.

El reto principal fue integrar una experiencia visual de alto impacto sin hacer pesada la navegación. También fue necesario conectar contenido de Instagram y carruseles de piezas sin romper la consistencia estética de la marca.

## Tecnologías utilizadas

1. Astro.js
2. React
3. Tailwind CSS
4. TypeScript
5. Swiper
6. API de Instagram

## Resultado

El sitio presenta colecciones, piezas personalizadas, contacto por WhatsApp e Instagram y una experiencia clara para explorar la propuesta de Ads Atelier.
