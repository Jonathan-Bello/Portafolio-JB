Volta Capital es un sitio corporativo bilingue para un conjunto de empresas relacionadas con soluciones de energia. Fue uno de los proyectos mas recientes dentro de esta etapa de trabajo, ya desarrollado con los nuevos estandares de diseño y desarrollo que se han ido consolidando en IGHU.

No fue un sitio particularmente complicado, pero si uno que exigia sobriedad. En una pagina corporativa de energia, la interfaz no debe competir con el mensaje: debe ordenar, sostener y transmitir confianza.

## Desarrollo y reto tecnico

Desarrollé el sitio con Astro, React, Tailwind CSS, TypeScript e i18n. La experiencia incluia contenido institucional, version en español e ingles, empresas relacionadas, tarjetas, modales, carruseles y una estructura visual orientada a comunicar solidez corporativa.

Tambien fue un proyecto donde integré con mas naturalidad herramientas de inteligencia artificial como apoyo en el flujo de trabajo. No como sustituto del criterio tecnico, sino como una forma de acelerar partes repetitivas, revisar estructura y avanzar con mayor eficiencia en tareas de implementacion.

La iconografia siguio el estandar de Heroicons, y algunas interacciones se resolvieron con React cuando tenia sentido hacerlo. Para este punto, la combinacion Astro + Tailwind + componentes puntuales ya funcionaba como una base estable para construir rapido sin sacrificar orden.

## Tecnologias utilizadas

1. Astro.js
2. React
3. Tailwind CSS
4. TypeScript
5. i18n
6. Heroicons

## Resultado

Volta Capital quedo publicado en su dominio principal y funciona como presencia corporativa bilingue. Para mi representa una etapa de madurez operativa: proyectos mas rapidos, decisiones mas claras y una forma de construir donde la herramienta ya no estorba, sino que acompaña.
