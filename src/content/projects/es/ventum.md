Ventum fue un proyecto Gatsby colaborativo en el que mi participación fue de mantenimiento y ajustes sobre una base ya existente.

## Desarrollo y reto técnico

Trabajé cambios puntuales en el sitio, cuidando no romper una implementación previa con Gatsby, React, Styled Components y Ant Design.

El reto de mantenimiento fue conservar consistencia en contacto, formularios, enlaces comerciales, tracking y secciones del sistema. Este tipo de proyecto exige entender código ajeno antes de modificarlo.

## Tecnologías utilizadas

1. Gatsby
2. React
3. Styled Components
4. Ant Design
5. React Calendar
6. React Helmet
7. Google Analytics
8. Gatsby Sitemap

## Resultado

El sitio se mantuvo operativo como canal informativo y comercial. Para portafolio, representa experiencia de mantenimiento profesional en proyectos heredados.
