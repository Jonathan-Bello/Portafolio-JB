Dirección Humana combina un sitio institucional con módulos operativos para vacantes y postulaciones. Es uno de los proyectos Gatsby donde mi participación fue más técnica.

## Desarrollo y reto técnico

Implementé módulos relacionados con vacantes, postulaciones y administración. La complejidad no estaba solo en pintar páginas, sino en conectar formularios, datos, cargas de información y flujos de revisión.

El reto técnico fue hacer convivir Gatsby con datos dinámicos desde Firebase, funciones serverless de Netlify, exportación a hojas de cálculo y estructuras útiles para SEO de ofertas laborales.

## Tecnologías utilizadas

1. Gatsby
2. React
3. Firebase
4. Firebase Admin
5. Netlify Functions
6. Styled Components
7. Formik y Yup
8. XLSX
9. Google Map React

## Resultado

El proyecto quedó como un sitio con valor operativo real: publicación de vacantes, recepción de postulaciones, administración de datos y soporte a procesos internos de reclutamiento.
