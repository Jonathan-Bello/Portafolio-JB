## Contexto

Este backend forma parte de la línea técnica de mi tesis y proyectos educativos relacionados con CSS.

## Función

La API recibe código CSS, lo procesa, consulta herramientas de análisis y prepara una respuesta que puede ser usada por el videojuego para orientar al estudiante.

## Importancia

El proyecto demuestra mi capacidad para conectar frontend, videojuegos, análisis de código e inteligencia artificial en una arquitectura útil para educación.
