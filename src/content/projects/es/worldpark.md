Worldpark es un sitio Gatsby para presentar desarrollos industriales, naves, áreas comerciales y ubicaciones estratégicas.

## Desarrollo y reto técnico

Participé en componentes visuales e interactivos, especialmente relacionados con sliders, catálogos y presentación de desarrollos.

El reto técnico fue manejar un sitio con volumen real de contenido: páginas por desarrollo, mapas, PDFs en Google Drive, videos desde Cloudinary y componentes reutilizables para mantener la información organizada.

## Tecnologías utilizadas

1. Gatsby
2. React
3. Styled Components
4. Bootstrap
5. Google Map React
6. Swiper
7. Cloudinary
8. Gatsby Sitemap

## Resultado

El sitio funciona como plataforma de presentación comercial para desarrollos industriales, con catálogos, mapas y contenido visual de apoyo.
