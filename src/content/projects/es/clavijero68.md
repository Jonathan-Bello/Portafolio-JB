Clavijero68 fue una landing comercial construida con Gatsby. El objetivo fue presentar información clave del proyecto, organizar secciones de valor y facilitar contacto desde una experiencia simple.

## Desarrollo y reto técnico

Trabajé la maquetación y organización de componentes de sección, integrando estilos con Sass, EDgrid, sliders, formulario y tracking.

El reto técnico estuvo en resolver una landing completa con una estructura limpia y reutilizable. Aunque no era un sistema complejo, requería cuidar responsive, jerarquía visual y puntos de contacto.

## Tecnologías utilizadas

1. Gatsby
2. React
3. Sass
4. EDgrid
5. Swiper
6. Formspree
7. Google Tag Manager

## Resultado

El sitio funciona como una landing estática, rápida y orientada a conversión, útil para mostrar trabajo con Gatsby en proyectos comerciales de entrega ágil.
