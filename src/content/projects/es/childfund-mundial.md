Este proyecto fue una landing para ChildFund México enfocada en captación de leads para apadrinamiento.

## Desarrollo y reto técnico

Desarrollé la landing y el flujo de registro, integrando formulario, API interna y envío de notificaciones por correo.

El reto fue usar Astro no solo como sitio estático, sino como proyecto con salida serverless en Netlify para recibir leads y enviar correos mediante Nodemailer.

## Tecnologías utilizadas

1. Astro.js
2. Tailwind CSS
3. Netlify adapter
4. Nodemailer

## Resultado

La landing presenta la campaña, guía al usuario hacia el registro y automatiza la comunicación interna cuando llega un nuevo lead.
