Tycoon es un sitio corporativo grande relacionado con licencias de marcas. El proyecto contiene páginas institucionales, noticias, vacantes y un catálogo amplio de marcas por región.

## Desarrollo y reto técnico

Participé en mantenimiento y ajustes del proyecto sobre una base Gatsby colaborativa. El sitio tiene muchas rutas y componentes asociados a marcas, noticias y contenido regional.

El reto técnico estuvo en trabajar dentro de un proyecto grande sin romper estructura, rutas ni contenido existente. También se detecta uso de internacionalización, tracking e integración con Instagram Graph API.

## Tecnologías utilizadas

1. Gatsby
2. React
3. Styled Components
4. Bootstrap
5. i18next
6. Gatsby React i18next
7. Instagram Graph API
8. Google Analytics

## Resultado

El proyecto muestra experiencia en mantenimiento de un Gatsby de mayor escala, con muchas páginas, catálogo corporativo, noticias e internacionalización.
