Zahreen fue una landing page desarrollada dentro del dominio principal de Basmar Beauty para comunicar una propuesta comercial específica.

## Mi rol

Desarrollé la landing, adapté la estructura visual al producto y preparé la publicación dentro de una ruta específica del dominio.

El reto fue trabajar con una base de URL controlada, manteniendo assets, enlaces y navegación funcionando correctamente dentro de `/zahreen/`.

La página funciona como experiencia comercial enfocada en lectura rápida, marca y conversión desde dispositivos móviles.
