Este sitio documenta el Game Design Document de CSS, un proyecto vinculado con videojuegos educativos, aprendizaje de CSS y herramientas web.

## Desarrollo y reto técnico

Desarrollé el sitio documental y conecté la experiencia web con enlaces a demo, repositorios, backend y configuración consumida por el juego.

El reto fue presentar documentación técnica de juego de forma navegable y útil, manteniendo relación clara entre web, Godot y backend.

## Tecnologías utilizadas

1. Astro.js
2. React
3. Tailwind CSS
4. JavaScript
5. Godot
6. Render

## Resultado

El sitio funciona como hub documental del proyecto, con acceso a materiales clave y una presentación web clara para explicar el sistema.
